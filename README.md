# Microfront usando Single SPA

Configuración raíz del proyecto microfrontend con single-spa.

## Requisitos previos

Antes de levantar este proyecto, es necesario que los siguientes microfrontends estén ejecutándose:

1. **Proyecto React**: `npm run start`
2. **Proyecto Vue**: `npm run serve`

## Inicio del proyecto

Una vez que los proyectos React y Vue estén levantados, ejecuta:

```bash
npm run start
```

El proyecto se ejecutará en el puerto 9000.
